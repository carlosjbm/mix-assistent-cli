# Skill UX-UI

## This skill helps enhance the user experience and the application interface.

## Rules

1. This skill should be applied whenever the user requests UI/UX improvements

2. Make the commands intuitive, provide clear visual feedback, and offer accessible help within the CLI itself. This turns a technical tool into a more user‑friendly and efficient experience.
